# TalkBalancer v0.3 販促・取扱説明画像

TalkBalancer の導入メリットから実際の使い方、通知、レポート、プライバシー、機材構成までを説明する8枚組の画像資料です。

1. `01-quick-start.png` — 表紙と3ステップ概要
2. `02-value-scenes.png` — 利用シーンと導入メリット
3. `03-start-consent.png` — 開始前宣言・同意・モード選択
4. `04-live-operation.png` — テーブル表示と幹事リモコン
5. `05-alert-catalog.png` — 9種の丁寧な通知
6. `06-report.png` — 終了レポートの見方
7. `07-privacy-logic.png` — 保存しない設計と削除フロー
8. `08-hardware-setup.png` — 推奨機材と設置方法

画像は横長16:9で、プレゼン表示・Web掲載・印刷配布を想定しています。

外部マイクが手元にない場合は、アプリの「マイク接続確認」からPC内蔵マイク簡易モードを利用できます。精度は下がるため、表示される0〜100とdBFSは同じPC・同じ配置で比較する参考値として扱います。

Android携帯では `/talkbalancer/mobile` をホーム画面へ追加し、携帯内蔵マイク・テーブル表示・幹事操作を統合した「携帯1台モード」として利用できます。

PWA起動時はブラウザのタブやアドレスバーが隠れます。さらに「表示専用」を押すと、操作UIが5秒後に隠れ、画面タップで再表示できます。
